Reference Guide

HTML

HTML boilerplate:

class="code"><!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Document</title> </head> <body> </body> </html>

In Visual Studio Code you can type ! + enter in an .html file to add a boilerplate

HTML Tags

Tag Name Description
<!DOCTYPE> HTML Doctype Declaration A HTML file must start with a Doctype declaration.
<a></a> Anchor attribute This is used to insert a link.
<body></body> Body This is the main section of the page. Everything within the body tags is visible in the browser.
<br> Break Break is used to enter whitepace in between 2 paragraphs.Break is a self-closing tag
<button></button> Button Adds a button to the page.
<datalist></datalist> Body This is the main section of the page. Everything within the body tags is visible in the browser.
<div></div> Div A div is a section which is styled with CSS.
<embed></embed> Embed With Embed you can embed other media like video or audio on a page. media wil be retrieved from the source page.
<footer></footer> Footer This is used to make the footer of a website. A footer is placed on the bottom of a website.
<head></head> Head This is the head section of a page. Everything within the head tags is not visible in the browser. the head tags are used for metadata.
<img> Image Image tags wil display an image.
